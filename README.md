# ⚡ Catalyst Elite v8.5
A lightweight, high-performance anime discovery engine packed into a single HTML file. No frameworks, no node_modules, no bloat. For those wondering why it's v8.5, it just sounds 😎 cool. https://skidieup.github.io/catalyst-anime-catalog/

# 👨‍💻 The Tech
Logic: Vanilla JS (ES6+)
API: Jikan v4 (MAL Open Source)
Styling: Tailwind CSS (CDN)
Interface: Optimized for mobile touch gestures.
# 💨 Quick Start
Drop index.html into your server.
Open it on a phone.
Done.
Note: This build uses a hardware detector. If you're on a desktop, use "Inspect Element" mobile view or you'll be blocked.
